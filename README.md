<h2 class="header2">Description</h2>
<p class="text-content">
Second Capstone project, a snake game made in the turtle graphics package. Small, simplistic and stylized to be retro or old school. Plays like snake, touch the walls or the body and you die, records the score and highscore of that session.

<h2>Prerequisites</h2>

<h4>Languages Used</h4>
- Python

<h4>API</h4>

<h4>Packages</h4>
- turtle<br>
- time

<h2>Installation</h2>

First clone this repo

```sh
git clone
```

Install to get the dependancies

```sh
python3 -m pip install -r requirements.txt
```

Start server / run terminal

```sh
Start server
```
